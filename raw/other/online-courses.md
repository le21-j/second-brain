# Online courses catalog — Physical-Layer ML Roadmap

Every video/interactive course cited in the [[python-ml-wireless]], with links and what chapters matter.

## Python

- **CS50P — Harvard Intro to Python** — https://cs50.harvard.edu/python/. Ten weekly problem sets, in-browser grading. Compressible to 3–4 weeks. Free.
- **Corey Schafer's Python tutorials** — https://www.youtube.com/c/Coreyms. YouTube. Decorators, generators, context managers, venvs, OOP.
- **Exercism Python track** — https://exercism.org/tracks/python. Daily drill with mentored code review.

## NumPy / SciPy

- **Stanford CS231n Python/NumPy tutorial** — https://cs231n.github.io/python-numpy-tutorial/. One dense hour, 80% coverage.
- **Rougier's 100 NumPy Exercises** — https://github.com/rougier/numpy-100. Drill source.
- **SciPy signal tutorial** — https://docs.scipy.org/doc/scipy/tutorial/signal.html.

## PyTorch

- **PyTorch 60 Minute Blitz** — https://pytorch.org/tutorials/beginner/deep_learning_60min_blitz.html.
- **Bourke "Learn PyTorch for Deep Learning"** — https://www.learnpytorch.io/. Free online book.
- **Raschka "PyTorch in One Hour"** — https://sebastianraschka.com/teaching/pytorch-1h/.

## Karpathy — non-negotiable

- **Neural Networks: Zero to Hero** — https://www.youtube.com/playlist?list=PLAqhIrjkxbuWI23v9cThsA9GvCAUhRvKZ. Do the **entire playlist**.
- **"Let's build GPT from scratch"** — https://www.youtube.com/watch?v=kCc8FmEb1nY.
- **nanoGPT** — https://github.com/karpathy/nanoGPT.
- **The Annotated Transformer** — http://nlp.seas.harvard.edu/annotated-transformer/.

## Classical ML

- **Andrew Ng — ML Specialization (2022 rebuild)** — https://www.coursera.org/specializations/machine-learning-introduction. Three courses. Python. Free audit.
- **Stanford CS229 (2018 Autumn)** — https://www.youtube.com/playlist?list=PLoROMvodv4rMiGQp3WXShtMGgzqpfVfbU. Rigor. Priority lectures: 2–10, 14–15, 17–20.
- **StatQuest — Josh Starmer** — https://www.youtube.com/@statquest. Just-in-time intuition.

## Deep learning

- **Andrew Ng — Deep Learning Specialization** — https://www.coursera.org/specializations/deep-learning. Five courses. Course 3 (Structuring ML Projects) is underrated.
- **Stanford CS231n — CNNs for Visual Recognition** — https://cs231n.stanford.edu/. **Do Assignment 2 fully**.
- **MIT 6.S191 — Intro to Deep Learning** — http://introtodeeplearning.com/.
- **NYU DS-GA 1008 — DL (LeCun + Canziani)** — https://atcold.github.io/NYU-DLSP20/. Energy-based, self-supervised.
- **Stanford CS224W — ML with Graphs (GNN)** — https://web.stanford.edu/class/cs224w/.
- **Stanford CS224N — NLP with DL** — https://web.stanford.edu/class/cs224n/.

## Reinforcement learning

- **David Silver — DeepMind/UCL RL Course** — YouTube playlist. Chalk-talk version of Sutton & Barto.
- **Hugging Face Deep RL Course** — https://huggingface.co/learn/deep-rl-course/.
- **OpenAI Spinning Up** — https://spinningup.openai.com/. Algorithm reference with pseudocode + PyTorch impls.
- **CleanRL** — https://github.com/vwxyzjn/cleanrl. Single-file reference implementations.
