# An Introduction to Statistical Learning (with Applications in Python) — James, Witten, Hastie, Tibshirani, Taylor

**Category:** Machine Learning (classical, with Python code)
**Status:** FREE PDF + Python labs
**URL:** https://hastie.su.domains/ISLP/ISLP_website.pdf
**Python package:** https://github.com/intro-stat-learning/ISLP
**Publisher:** Springer, 2023
**Roadmap phase:** Phase 1 Month 1 — primary classical-ML textbook

## Topic coverage (chapters 2–10, 12)
- Ch 2: Statistical learning foundations (bias-variance, loss functions)
- Ch 3: Linear regression
- Ch 4: Classification (logistic regression, LDA, QDA, naive Bayes)
- Ch 5: Resampling (cross-validation, bootstrap)
- Ch 6: Regularization (ridge, lasso, best-subset)
- Ch 7: Non-linear regression (splines, GAMs, local regression)
- Ch 8: Trees, bagging, random forests, boosting, BART
- Ch 9: SVMs and kernels
- Ch 10: Deep learning (CNNs, RNNs — basic)
- Ch 12: Unsupervised (PCA, clustering)

## Why it's on the roadmap
ISLP is the Python-native rewrite of ISL (which used R). Pedagogically gentle, minimally mathematical, heavy on conceptual understanding — exactly the texture you want when building intuition. Andrew Ng's Coursera specialization is the "watch a video" form; ISLP is the "sit down and read" form.

- Stanford CS229 for rigor once ISLP is done.
