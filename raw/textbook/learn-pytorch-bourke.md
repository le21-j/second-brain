# Learn PyTorch for Deep Learning — Daniel Bourke

**Category:** PyTorch (free online book; code-first)
**Status:** FREE online book
**URL:** https://www.learnpytorch.io/
**GitHub:** https://github.com/mrdbourke/pytorch-deep-learning
**Companion videos:** https://www.youtube.com/@mrdbourke — full playlist mirrors the book chapter-by-chapter
**Author:** Daniel Bourke (ML engineer, course creator)
**Roadmap phase:** Phase 1 Month 3 (PyTorch fluency)

## Topic coverage
- **00 PyTorch Fundamentals** — tensors, tensor ops, GPU moves.
- **01 PyTorch Workflow** — full training loop from first principles.
- **02 Neural Network Classification** — binary + multi-class classification.
- **03 Computer Vision** — CNNs on CIFAR / FashionMNIST.
- **04 Custom Datasets** — `Dataset` / `DataLoader` patterns (critical for wireless data: this is how you load DeepMIMO / DeepSense later).
- **05 Going Modular** — factor notebook code into scripts.
- **06 Transfer Learning** — fine-tuning pretrained models (the pattern [[large-wireless-model]] uses).
- **07 Experiment Tracking** — TensorBoard + Weights & Biases.
- **08 Paper Replicating** — reproducing the ViT paper from scratch.
- **09 Model Deployment** — Gradio + Hugging Face Spaces.

## Why it's on the roadmap
The roadmap lists this *alongside* the 60 Minute Blitz as a first-pass PyTorch resource:
> "Start with the official 60 Minute Blitz, then the 'Learn PyTorch for Deep Learning' free online book by Daniel Bourke."

Bourke's signature is **heavy repetition with small variation** — each chapter rebuilds the same training-loop skeleton until it's muscle memory. That's exactly what a Phase-1 learner needs.

## Where it fits vs the other PyTorch resources
- **Bourke (this book)** — hands-on, tutorial-style, for first fluency.
- **[[deep-learning-with-pytorch]] (Stevens/Antiga/Viehmann)** — internals (tensors, storage, strides, autograd).
- **[[ml-with-pytorch-scikit-learn]] (Raschka)** — sklearn → PyTorch bridge.
- **Karpathy Zero-to-Hero** — build everything from scratch; deepest pedagogy.

Do Bourke first to reach fluency quickly, then Stevens for depth, then Karpathy for the first-principles rebuild.
