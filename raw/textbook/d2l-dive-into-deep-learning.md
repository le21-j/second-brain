# Dive into Deep Learning (D2L) — Zhang, Lipton, Li, Smola

**Category:** Deep Learning (code-first companion)
**Status:** FREE (CC BY-SA 4.0)
**URL:** https://d2l.ai/
**GitHub:** https://github.com/d2l-ai/d2l-en
**Authors:** Aston Zhang, Zachary Lipton, Mu Li, Alex Smola (AWS, CMU)
**Roadmap phase:** Phase 1 Month 4 → Phase 2; pair with Prince

## Topic coverage
- Part 1: Preliminaries (NumPy, calculus, probability — thin).
- Part 2: Linear networks, MLPs.
- Part 3: Deep learning computation (layers, modules, training, GPU).
- Part 4: CNNs (LeNet, AlexNet, VGG, ResNet, DenseNet).
- Part 5: Sequence models (RNN, GRU, LSTM).
- Part 6: Attention and transformers.
- Part 7: Optimization algorithms.
- Part 8: Computational performance (distributed, CPU vs GPU).
- Part 9: Computer vision.
- Part 10: NLP.
- Part 11: Generative adversarial networks.
- Part 12: Recommender systems.

## Why it's on the roadmap
> "The best code-first companion. Read the concept in Prince, run the code in d2l."

Every section has working code in **MXNet, PyTorch, TensorFlow, and JAX** — you can pick your backend. Written by Amazon researchers so there's no hidden framework preference. Includes exercises that actually train on your laptop.
