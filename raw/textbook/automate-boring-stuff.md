# Automate the Boring Stuff with Python — Al Sweigart

**Category:** Python fundamentals
**Status:** free online + paid print — not yet in repo
**URL:** https://automatetheboringstuff.com/
**Roadmap phase:** Phase 1 weeks 1–2 (momentum builder)

## Topic coverage
- Part I: Python basics (control flow, functions, lists, dicts, strings, regex)
- Part II: Automating common tasks — web scraping, Excel, PDF, email, scheduling, GUIs, image manipulation

## Why it's on the roadmap
The roadmap calls this out as "excellent for building early momentum" — tasks are immediately useful (renaming files in bulk, scraping a page) so the reward signal is high. The book is licensed CC BY-NC-SA, so reading it online is legitimate.

## What to skip for this roadmap
Parts on web scraping, GUI, email automation are cute but not load-bearing for ML or wireless work. Read Part I + the regex chapter, skim Part II.
