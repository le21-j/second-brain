# Python for Data Analysis (3rd ed.) — Wes McKinney

**Category:** Scientific Python (pandas)
**Status:** FREE online (author-authorized open edition)
**URL:** https://wesmckinney.com/book/
**Roadmap phase:** Phase 1 weeks 3–4 (pandas exposure)

## Topic coverage
- NumPy refresher
- pandas Series and DataFrame
- Data loading, cleaning, merging, groupby, pivot, time series
- Visualization via matplotlib, seaborn
- Case studies: MovieLens, US Baby Names

## Why it's on the roadmap
Wes McKinney **wrote pandas**; he knows exactly which API corners matter. Pair with the official "10 minutes to pandas" for a compressed on-ramp. For ML+wireless research, pandas is used mostly for logging and result summarization — you don't need deep pandas fu, but you need enough to build result tables cleanly.

## Minimum for this roadmap
- Chapters 5–8 (Series/DataFrame, data wrangling, join/concat).
- Chapter 11 (time series) if you're doing any scheduling/link-adaptation work.
