# Deep Learning with PyTorch — Stevens, Antiga, Viehmann

**Category:** PyTorch (internals + practice)
**Status:** FREE PDF from Manning (via form)
**URL:** https://www.manning.com/books/deep-learning-with-pytorch
**Authors:** Eli Stevens, Luca Antiga, Thomas Viehmann
**Publisher:** Manning, 2020
**Roadmap phase:** Phase 1 Month 3, Phase 2

## Topic coverage
- Part 1: Foundations (PyTorch's tensor, dtypes, storage, strides, views, autograd, optimizer)
- Part 2: Real-world projects — a lung-cancer CT detector in end-to-end detail (data loading → training → deployment)
- Part 3: Deployment

## Why it's on the roadmap
The roadmap calls out **chapters 1–8 specifically**: "explain PyTorch internals (tensors, storage, strides, autograd) better than any other source." This is true — Stevens et al. walk through PyTorch's memory model, which matters when you later debug a Sionna tutorial or profile GPU utilization.

## Chapters that matter most
1. Ch 1–3: tensor + NumPy analogues.
2. Ch 4: real-world tensor examples (time-series, text, images) — good "how do I load my data" reference.
3. Ch 5: autograd mechanics.
4. Ch 6–7: first neural-network, first CNN.
5. Ch 8: convolutions in depth.
