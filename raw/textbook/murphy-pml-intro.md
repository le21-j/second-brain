# Probabilistic Machine Learning: An Introduction — Kevin Murphy

**Category:** Machine Learning (encyclopedic, probabilistic)
**Status:** FREE (CC BY-NC-ND)
**URL:** https://probml.github.io/pml-book/book1.html
**Author:** Kevin Murphy (Google DeepMind)
**Publisher:** MIT Press, 2022
**Roadmap phase:** reference, not linear read

## Topic coverage
- Part I: Foundations (probability, statistics, linear algebra, optimization)
- Part II: Linear models, deep learning basics
- Part III: Beyond (clustering, dimensionality reduction, GNNs, graphical models, RL)
- Part IV: Beyond supervised (generative models, structured prediction)

Murphy also wrote the 2012 textbook of the same name; this is a full rewrite. A second volume (**Advanced Topics**, https://probml.github.io/pml-book/book2.html) adds graphical models, causality, advanced generative models.

## Why it's on the roadmap
> "The modern encyclopedic replacement [for Bishop] — use it as reference, not a linear read."

The pedagogy is excellent; the scope is enormous; **nobody reads this cover to cover**. Keep the HTML open in a tab and CTRL-F for the thing you need.
