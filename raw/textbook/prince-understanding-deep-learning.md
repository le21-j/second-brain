# Understanding Deep Learning — Simon J. D. Prince

**Category:** Deep Learning (primary textbook for this roadmap)
**Status:** FREE (CC BY-NC-ND)
**URL:** https://udlbook.github.io/udlbook/
**Publisher:** MIT Press, 2023 (hardcover paid; online PDF free)
**Author:** Simon Prince (Computer Vision, formerly Borealis AI / RBC)
**Roadmap phase:** Phase 1 Month 4 → Phase 2 end; **primary DL text**

## Topic coverage
- Ch 1–2: Supervised learning basics
- Ch 3–4: Shallow → deep networks
- Ch 5: Loss functions (MSE, cross-entropy, likelihood framing)
- Ch 6: Training (SGD, Adam, initialization, batchnorm)
- Ch 7: Gradients and initialization
- Ch 8: Measuring performance (bias-variance for DL)
- Ch 9: Regularization (weight decay, dropout, early stopping)
- Ch 10: **Convolutional networks** — the best exposition available
- Ch 11: Residual networks
- Ch 12: Transformers — clearest transformer exposition in any book
- Ch 13–14: Graph NNs, unsupervised
- Ch 15: Generative Adversarial Networks
- Ch 16: Normalizing flows
- Ch 17: Variational Autoencoders
- Ch 18: Diffusion models
- Ch 19: Reinforcement learning
- Ch 20: Why does deep learning work? (deep learning theory)

## Why it's on the roadmap
> "The modern successor to Goodfellow et al., with the best pedagogical figures available. **This is your main DL book.**"

Prince's figures are genuinely excellent — he spent years making them. Readers consistently report understanding concepts from Prince that Goodfellow didn't make click. For transformers (Ch 12) and diffusion (Ch 18) specifically, this is the clearest published treatment.

## How to use it with this roadmap
- Read the chapter in Prince → run the corresponding notebook in [[d2l-dive-into-deep-learning]] → attempt a Kaggle/CS231n problem that uses it.
- For architecture-specific depth (CNNs, transformers) cross-reference Karpathy's videos.
