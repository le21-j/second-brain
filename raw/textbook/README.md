# Textbook shelf — reading list for the Physical-Layer ML Roadmap

This shelf doesn't yet contain the book PDFs themselves — it contains **reference cards** for each book on Jayden's 14-month roadmap ([[python-ml-wireless]]). Each card captures: title, author/edition, where to get it (most are free), which chapters matter, and what topics it covers. When Jayden actually reads a chapter, he drops the PDF/notes into `raw/` and the LLM ingests at that point.

Organized by track.

## Python

- [python-crash-course.md](python-crash-course.md) — Matthes, 3rd ed. Fastest on-ramp.
- [automate-boring-stuff.md](automate-boring-stuff.md) — Sweigart. Free. Early-momentum book.
- [fluent-python.md](fluent-python.md) — Ramalho, 2nd ed. Definitive intermediate/advanced Python.

## Scientific Python

- [from-python-to-numpy.md](from-python-to-numpy.md) — Rougier. Free, online. Best vectorization resource.
- [scientific-visualization-matplotlib.md](scientific-visualization-matplotlib.md) — Rougier. Free. Publication-quality figures.
- [pandas-mckinney.md](pandas-mckinney.md) — Wes McKinney "Python for Data Analysis". Free online.

## Wireless / DSP

- [pysdr-lichtman.md](pysdr-lichtman.md) — Marc Lichtman. Free online textbook. The single most wireless-relevant free resource anywhere. OFDM, ML-for-RF chapter.
- [ml-and-wireless-comm-eldar.md](ml-and-wireless-comm-eldar.md) — Eldar/Goldsmith/Gündüz/Poor, Cambridge 2022. Current academic reference.
- [simeone-ml-for-engineers.md](simeone-ml-for-engineers.md) — Osvaldo Simeone. Engineer-oriented ML primer.

## PyTorch

- [learn-pytorch-bourke.md](learn-pytorch-bourke.md) — Daniel Bourke. FREE online book. First-fluency, tutorial-style.
- [deep-learning-with-pytorch.md](deep-learning-with-pytorch.md) — Stevens/Antiga/Viehmann. Free PDF from Manning. Ch 1–8 explain PyTorch internals best.
- [ml-with-pytorch-scikit-learn.md](ml-with-pytorch-scikit-learn.md) — Raschka. Code free on GitHub. Bridges sklearn→PyTorch.

## TensorFlow / Keras

- [deep-learning-with-python-chollet.md](deep-learning-with-python-chollet.md) — François Chollet, 2nd ed. Authoritative Keras source. Ch 3, 7, 9 cover what's needed for Sionna.

## Machine Learning (classical)

- [islp-introduction-to-statistical-learning.md](islp-introduction-to-statistical-learning.md) — ISLP, Python ed. Free PDF. Chapters 2–10, 12.
- [bishop-prml.md](bishop-prml.md) — Christopher Bishop, "Pattern Recognition and Machine Learning". The most valuable reference for a DSP-trained applicant. Ch 1–4, 6, 8, 9–11, 13.
- [murphy-pml-intro.md](murphy-pml-intro.md) — Kevin Murphy, "Probabilistic Machine Learning: An Introduction". Free. Modern encyclopedic.

## Deep Learning

- [prince-understanding-deep-learning.md](prince-understanding-deep-learning.md) — Simon Prince. Free. **Primary DL textbook** for this roadmap.
- [d2l-dive-into-deep-learning.md](d2l-dive-into-deep-learning.md) — d2l.ai. Free. Code-first companion to Prince.
- [goodfellow-deep-learning.md](goodfellow-deep-learning.md) — Goodfellow/Bengio/Courville. Free. Theoretical reference.

## Reinforcement Learning

- [sutton-barto-rl.md](sutton-barto-rl.md) — Sutton & Barto, 2nd ed. Free. The RL canon.

## Math

- [matrix-calculus-for-deep-learning.md](matrix-calculus-for-deep-learning.md) — Parr & Howard. Free, one-sitting.
- [mackay-information-theory.md](mackay-information-theory.md) — David MacKay, "Information Theory, Inference, and Learning Algorithms". Free. Ch 47–50 on LDPC/turbo/fountain are the wireless-relevant gems.
- [boyd-convex-optimization.md](boyd-convex-optimization.md) — Boyd & Vandenberghe. Free. Ch 1–5.
