# Machine Learning with PyTorch and Scikit-Learn — Sebastian Raschka

**Category:** ML + PyTorch bridge
**Status:** paid book; **code is free on GitHub**
**Code:** https://github.com/rasbt/machine-learning-book
**Publisher:** Packt, 2022
**Author:** Sebastian Raschka (Lightning AI, formerly U. Wisconsin)
**Roadmap phase:** Phase 1 Month 3 companion

## Topic coverage
- Classical ML (linear/logistic regression, SVM, decision trees, ensemble) in scikit-learn
- Data preprocessing, model evaluation, hyperparameter tuning
- Intro to deep learning in PyTorch (MLPs, CNNs, RNNs, transformers)
- Generative models (VAEs, GANs)
- Graph neural networks
- Reinforcement learning

## Why it's on the roadmap
Raschka's signature move is clarity: every chapter is a minimal working example. The book is ideal for the transition from "I know sklearn" to "I can write a PyTorch training loop." Pair it with Raschka's **"PyTorch in One Hour"** refresher at https://sebastianraschka.com/teaching/pytorch-1h/.
