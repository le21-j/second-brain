# Conjugate Symmetry

## In one line
If $x[n]$ is **real**, then $X[N-k] = X^*[k]$ — the back half of the spectrum is just the complex conjugate of the front half.

## Example first
You compute the DFT of real $x[n] = [1, 2, 0, 0]$ ($N=4$). You get $X = [3, 1-2j, -1, 1+2j]$. Check:
- $X[N-1] = X[3] = 1 + 2j$
- $X^*[1] = (1 - 2j)^* = $ **$1 + 2j$** $\checkmark$
- $X[N-2] = X[2] = -1$
- $X^*[2] = (-1)^* = $ **$-1$** $\checkmark$ (real, self-conjugate at $N/2$)

So $X[2]$ and $X[3]$ contain **no new info** — they're determined by $X[0]$ and $X[1]$. Only **$N/2 + 1$** bins are independent for a real input.

## The idea
Start from the definition:
$$X[N-k] = \sum_n x[n]\, e^{-j2\pi(N-k)n/N} = \sum_n x[n]\, e^{+j2\pi kn/N}$$
If $x[n]$ is real, complex-conjugating the sum is the same as flipping the sign in the exponent:
$$X^*[k] = \sum_n x[n]\, e^{+j2\pi kn/N} = X[N-k]$$
Done. **For real $x$, $X[N-k] = X^*[k]$.**

A useful equivalent form (replace $k$ with $k + N/2$):
$$X[N/2 + k] = X^*[N/2 - k]$$
— the spectrum is **conjugate-symmetric about the Nyquist bin**.

## Why it matters
- **Storage:** to store the DFT of a real $N$-point signal you only need $N/2 + 1$ complex values (bins $0$ through $N/2$). The rest can be regenerated by conjugation.
- **[[real-valued-fft]]:** lets you compute $N$-point real FFT using an $N/2$-point complex FFT (halving the work).
- **Plotting:** don't double-count a real spectrum by plotting all $N$ bins. For a real signal, plot bins $0,\ldots,N/2$ only.

## Common mistakes
- Assuming the symmetry holds for complex $x[n]$ — it **doesn't**. Complex signals have asymmetric spectra.
- Applying it to $X[N/2]$. Since $N/2$ is its own "mirror" ($N - N/2 = N/2$), it must be **real** for a real input. Sometimes it's non-zero, sometimes zero, but never has an imaginary part.
- Forgetting that bin $0$ is also self-conjugate ($X[0] = X^*[0] \to$ real).
