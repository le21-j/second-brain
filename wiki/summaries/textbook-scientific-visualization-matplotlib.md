# Rougier — Scientific Visualization: Python + Matplotlib

**Status:** stub — free at https://github.com/rougier/scientific-visualization-book. Reference card at `raw/textbook/scientific-visualization-matplotlib.md`.

## TL;DR
**The open-access gold standard for publication-quality matplotlib figures.** Papers and GitHub READMEs live or die on their headline figure. Default matplotlib output is ugly; this book's patterns produce work good enough for Nature/Science-style print.

## The portfolio rule (the load-bearing claim of this book for Jayden)

> **Every Phase 2+ repo README must have a headline figure in the first screen.** That figure must be made with this book's patterns: clear typography, a named perceptually-uniform colormap (not the default), labeled axes, zero-waste whitespace, saved as SVG or 300-DPI PNG.

NVIDIA + Wi-Lab interviewers will look at GitHub READMEs in 60 seconds. The headline figure is what they remember.

## Topic coverage
- Anatomy of a matplotlib figure (Figure / Axes / Artist hierarchy).
- Layout, typography, color (perceptually uniform colormaps, colorblind-safe palettes).
- Common traps + their fixes.
- Gallery: 3D, animations, dashboards, network visualizations.
- SVG / 300-DPI export patterns.
