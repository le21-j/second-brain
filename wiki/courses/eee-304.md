# EEE 304 — Signals and Systems / Communication Systems

**Software:** MATLAB + Simulink (for lab work)

## Overview

Continuous-time signals and systems with an applied focus on **communication systems** — Fourier analysis, frequency-domain reasoning, modulation/demodulation, and filter design. The lab arc uses Simulink to build modulators and demodulators and observe spectra and waveforms in real time. Concepts here pair naturally with the FFT material in [[eee-404]] and the modulation-aware ML material in [[modulation-classification]] / [[autoencoder-phy]] on the [[python-ml-wireless]] track.

## Roadmap

### Amplitude Modulation arc
1. **[[amplitude-modulation]]** — $\Phi_{AM}(t) = (A + m(t))\cos(\omega_c t)$, why we shift to a high carrier, USB/LSB
2. **[[modulation-index]]** — $\mu = m_{\text{amp}} / A$; under / perfect / over-modulation
3. **[[coherent-demodulation]]** — multiply by carrier again + LPF; works for any $\mu$
4. **[[envelope-detection]]** — non-coherent: $|\text{signal}|$ + bandpass; breaks when $\mu > 1$
5. **[[butterworth-filter]]** — order vs cutoff trade-off; the workhorse LPF/BPF in this lab

### Pulse modulation + multiplexing arc (HW7)
6. **[[pulse-amplitude-modulation]]** — sample, then transmit each sample as a fixed-width pulse
7. **[[time-division-multiplexing]]** — interleave multiple PAM streams in a frame with a sync slot
8. **[[chopper-amplifier]]** — square-wave-modulate up, bandpass-amplify, demodulate down: $G = 2A\sin^2(\pi D)/\pi^2$

### Final Exam arc (added 2026-05-04)
The 8-problem final spans the whole course: steady-state response, filter design (CT/DT), ROC, Euler/Tustin discretization, PAM/TDM-FDM bandwidth, closed-loop control, sampled-data CT equivalents.

- [[steady-state-response]] — Problem 1: sinusoidal $y_{ss}(t) = |H(j\omega)|\sin(\omega t + \angle H(j\omega))$
- [[forward-euler-discretization]] — Problem 5/8: $z = 1 + sT$; can destabilize
- [[tustin-bilinear]] — Problem 5/8: $z = (1+sT/2)/(1-sT/2)$; safe + best frequency match
- [[feedback-control-loop]] — Problem 7: closed-loop TFs $T_{ry} = CP/(1+CP)$, $T_{dy} = P/(1+CP)$, FVT for $e_{ss}$
- [[sampled-data-system]] — Problem 8: AAF → sample → DT filter → reconstruct, reverse-engineer CT via FE or Tustin
