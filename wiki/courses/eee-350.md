# EEE 350 — Random Signal Analysis

**School:** Arizona State University
**Textbook reference (cited in slides):** Wiley textbook + MIT 6.041 (Tsitsiklis, Probabilistic Systems Analysis and Applied Probability, OCW).

## Overview
Probability and statistics as used in signal analysis: from covariance and multivariate distributions, through the asymptotic theorems (LLN, CLT), into inference (Bayesian + classical), and finishing with an intro to stochastic processes. This is the **late-semester** arc of the course.

## Roadmap (what's in the wiki so far)

### Arc 1 — Moments & dependence
- **[[covariance]]** — $\text{Cov}(X,Y)$
- **[[correlation-coefficient]]** — $\rho_{XY}$
- **[[variance-of-a-sum]]** — $\text{Var}(X+Y) = \text{Var}\,X + \text{Var}\,Y + 2\,\text{Cov}(X,Y)$
- **[[bivariate-gaussian]]**, **[[multivariate-gaussian]]**
- **[[random-vector]]**, **[[iid-samples]]**
- **[[max-of-iid]]** — $F_{\max}(t) = F(t)^n$
- **[[conditional-expectation]]** — $E[X \mid Y]$ is a random variable
- **[[iterated-expectations]]** — $E[E[X \mid Y]] = E[X]$
- **[[conditional-variance]]** + **[[law-of-total-variance]]**
- **[[sum-of-random-number-of-rvs]]** — $E[S] = E[N]\cdot E[X]$; variance has two terms

### Arc 2 — Asymptotic theorems
- **[[chebyshev-inequality]]** — Markov's cousin; tail bound using variance
- **[[convergence-in-probability]]** vs deterministic limit
- **[[weak-law-of-large-numbers]]** (WLLN) — sample mean $\to \mu$ in probability
- **[[central-limit-theorem]]** (CLT) — normalized sum $\to \mathcal{N}(0,1)$
- **[[standardization]]** — the $(X - \mu)/\sigma$ transformation; distinct from CLT
- **[[variance-scaling-rule]]** — $\text{Var}(cX) = c^2\cdot\text{Var}(X)$, the algebraic fact behind standardization
- **[[continuity-correction]]** — the $\pm 0.5$ when approximating a discrete distribution by CLT
- **[[binomial-via-clt]]** — the classic application

### Arc 3 — Inference
- **[[bayesian-inference]]** — prior + likelihood $\to$ posterior
- **[[detection-vs-estimation]]** — discrete $\theta$ vs continuous $\theta$
- **[[map-detection]]** — argmax posterior; minimizes prob. of error
- **[[map-estimation]]**, **[[lms-estimation]]** (conditional mean, = MMSE)
- **[[linear-mmse-estimation]]** — aX + b, the linear-restricted estimator (also called LLS / LMSE)
- **[[maximum-likelihood-estimation]]** (MLE) — no prior
- **[[unbiased-estimator]]**, **[[consistent-estimator]]**
- **[[confidence-interval]]**
- **[[neyman-pearson-test]]** — no prior; choose between $H_0/H_1$
- **[[significance-test]]** — the reject/fail-to-reject framework under a fixed $\alpha$
- **[[type-i-error]]** / **[[type-ii-error]]**
- **[[likelihood-ratio-test]]** (LRT)
- **[[chi-squared-test]]** — testing variance

### Arc 4 — Applied / descriptive
- **[[linear-regression]]** — least squares as MLE under Gaussian noise
- **[[power-law-regression]]** — take logs to linearize
- **[[sample-mean]]**, **[[sample-variance]]**, **[[sample-covariance]]**
- **[[sample-median]]**, **[[order-statistics]]**, **[[sample-mode]]**
- **[[histogram]]**

### Arc 5 — Stochastic processes (intro)
- **[[stochastic-process]]** — collection of RVs indexed by time
- **[[stationary-process]]** — joint distribution depends on relative times
- **[[white-gaussian-process]]**, **[[colored-noise]]**
- **[[poisson-process]]** — exponential interarrivals
- **[[markov-chain]]** — state transitions with memory
