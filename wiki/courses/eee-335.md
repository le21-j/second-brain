# EEE 335 — Analog and Digital Circuits

**School:** Arizona State University, Spring 2026
**Textbook:** Sedra & Smith, *Microelectronic Circuits* (8th edition)
**Lab tool:** Cadence (EECAD lab, also remote-accessible via Apporto)

## Overview
The integrated MOSFET-centric circuits course: MOS device physics → CMOS logic gates → memory cells → small-signal IC amplifiers → frequency response → multi-transistor + differential amplifiers. Lab arc uses Cadence to build and characterize the circuits taught in lecture. Heavily problem-set driven; expect Sedra/Smith chapter problems weekly.
